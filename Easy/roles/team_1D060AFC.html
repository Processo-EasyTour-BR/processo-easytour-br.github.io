    Main Description

    Uma equipe reúne membros, que como um todo, é capaz de realizar todas as várias tarefas para as quais ele assume coletivamente a responsabilidade durante cada sprint.
    É capaz de determinar o que precisa ser feito para alcançar o objetivo de sprint. Ele é organizado de forma autônoma e planeja suas próprias tarefas, sem interferência externa.

    Staffing
    Skills

    A equipe deve ser capaz de executar todas as várias tarefas para as quais ela se responsabiliza coletivamente durante o planejamento de sprint. Isso significa que ele deve ser multifuncional para cobrir análise, design, codificação, teste e outras tarefas.

    Assignment ApproachesUm Easy team é geralmente feito de 3 a 6 membros.
    Key Considerations

    Através de uma abordagem centrada na coletividade, a agilidade contribui para melhorar a comunidade dentro da equipe.
