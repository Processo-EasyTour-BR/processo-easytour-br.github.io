Main Description

Um Release é o resultado de uma série de Sprints e por último, em geral, por poucos meses. Durante o ciclo de vida do produto, muitos releases são lançados.

Um release tem os seguintes atributos:

  • Objetivo
  • Tipo, "fixo" ou "variável" (padrão). Se for "corrigido", é fornecida uma data. 
  • Tamanho, em dias, para as Sprints. 
  • Número previsto de sprints

Release planning é um nível de planejamento de projeto: mostra em um nível alto, a lista de itens a serem desenvolvidos durante este release' sprints.
