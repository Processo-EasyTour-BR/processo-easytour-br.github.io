Description
Main Description

A documentação do produto é a referência de requisitos. Contém itens de backlog (também chamados itens de backlog do produto).

A sua vida está vinculada à do produto.



Brief Outline

Visível por todos, é atualizado regularmente. É submetido à revisão do planejamento de sprint e, uma vez aprovado, a parte que contém os requisitos alocados para o sprint atual é congelada.
