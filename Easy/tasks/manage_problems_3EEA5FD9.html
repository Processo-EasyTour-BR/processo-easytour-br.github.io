Task: Manage problems
Tome em consideração os eventos que acontecem a cada momento em um projeto e tente resolvê-los.
Disciplines: Easy Tasks
Purpose

O objetivo é eliminar os problemas experimentados pela equipe para que ele possa se concentrar em seus objetivos reais.

      Steps
      Esteja ciente do problema

      A reunião diária é o momento ideal para detectar problemas com os quais uma equipe está lutando. Pode até permitir resolver imediatamente através da colaboração em equipe. Contudo:

      • Problemas urgentes podem aparecer entre 2 daily scrums
      • A resolução de problemas não é realizada durante a reunião (a menos que a solução seja trivial)
      Tomando decisões sobre o plano de ação

      Identificando possíveis soluções.
      Uma reunião com gerentes é agendada se problemas com o projeto não puderem ser resolvidos facilmente ou precisar de uma decisão de gerenciamento.

      O  Product Owner deve tentar resolver problemas o mais rápido possível, para não perturbar o progresso da equipe.

      Key Considerations

      Esta tarefa está focada na resolução de problemas.
