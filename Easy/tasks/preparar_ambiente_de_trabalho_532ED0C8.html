Task: Preparar Ambiente de Trabalho
Planejar os recursos que serão necessários para o trabalho prosseguir
Disciplines: Easy Tasks
Purpose
Planejar os recursos que serão necessários para o trabalho prosseguir
Relationships
RolesPrimary Performer:
  • Easy Team
Additional Performers:
  • Product Owner
Outputs
  • Documento de Ambiente Trabalho
    Main Description
    Planejar as tarefas e prever os recursos e o ambiente necessários, incluindo, por exemplo, equipamentos, ferramentas, serviços, componentes, viagens e requisitos de processo (processos especiais para o projeto). Todos os recursos precisam ser explicitamente planejados, mesmo os já considerados como existentes e disponíveis ou que serão compartilhados com outros projetos, uma vez que se trata da sua alocação para uso. Caso não haja necessidade de nenhum recurso a ser adquirido para o projeto deve-se registrar o fato de que a questão foi examinada.
