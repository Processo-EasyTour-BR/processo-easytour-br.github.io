Main Description
Dentro do Easy, existem três artefatos principais que são usados pelas funções para transmitir informações sobre o produto que está sendo construído, as tarefas que são necessárias para criar os recursos especificados e os incrementos de funcionalidade.

Esclarecendo claramente quais são esses artefatos de base e os proprietários dos artefatos estabelecerão uma compreensão dos papéis e suas responsabilidades no processo de gerenciamento do produto, projeto e processo.
Key Considerations

A lista de artefatos não é exaustiva, mas representa os componentes-chave para um processo Easy.
